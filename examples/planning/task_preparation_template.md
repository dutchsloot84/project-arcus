# Task Preparation Template

This template is a **non-canonical preparation aid** for work that is likely to become a real Arcus task later.

## Warning

- Do not place this file under `planning/`.
- Do not invent a task ID.
- Do not treat a filled-in copy as approved or committed work.
- Convert this content into canonical task YAML only after an explicit task ID is issued and the work is intentionally accepted into the planning control plane.
